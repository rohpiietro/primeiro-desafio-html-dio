História da web

Primeiros computadores

Antigamente os computadores eram gigantes e pesavam toneladas por serem eletromecânicos(1º geração), seu objetivo inicial era fazer cálculos. Na 2º Guerra mundial teve outro avanço expressivo, visto que as máquinas estavam se tornando mais úteis em tarefas de desencriptação de mensagens inimigas e criação de novas armas mais inteligentes. Na 2º geração, houve a substituição das válvulas eletrônicas por transístores com isso os computadores ficaram bem menores. Na 3º geração foi introduzido os circuitos integrados. Já a 4º geração foi introduzido o microprocessador e continua até os dias de hoje.

como surgiu a internet voltar

A internet é uma evolução da comunicação a distância. Tudo começou com o código morse através do telégrafo e após muitos anos de pesquisas foi criado a arpa NET, uma rede de computadores construída em 1969 para transmissão de dados militares sigilosos e no mesmo ano Tim Berners-Lee criou o primeiro navegador.

O que são Clients voltar

Um cliente web é qualquer componente de software que faz solicitações a um servidor web para acessar informações ou serviços disponibilizados na internet. Exemplos comuns de clientes web incluem navegadores como

Chrome, Firefox e Safari
, que solicitam páginas da web e exibem os conteúdos para os usuários. Na relação cliente-servidor, o cliente web atua como o ponto de interação do usuário, consumindo dados e serviços fornecidos por servidores hospedados em infraestrutura de hospedagem, como a oferecida por empresas como a Hostinger. A diferença entre aplicativo e aplicação web é que o aplicativo é um software que roda em um dispositivo móvel, enquanto a aplicação web é um software que roda em um navegador de internet

O que são Servers voltar

Um servidor é um sistema que processa, armazena dados e fornece serviços para outros computadores(clientes). Ele pode ser uma máquina física ou virtual na nuvem. Sua principal função é atender pedidos de dados e serviços, garantindo acesso rápido e eficiente.

Linguagens de Programação voltar

Linguagem de programação é um sistema de comunicação entre programadores e computadores, que permite a criação de programas e softwares. Server-side é um termo que se refere a operações que são realizadas no servidor, e por outro lado cliente-side é uma linguagem que é executada no lado cliente, ou seja, no computador do próprio usuário. O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação.

Introdução ao HTML na Prática voltar

No HTML usa essas três tags: A tag html define todo o documento HTML. A tag head contém metadados sobre o título e a codificação de caracteres da página. A tag body inclui todo o conteúdo que aparece na página. A expressão “tag” significa literalmente “etiqueta”. Uma etiqueta é usada para marcar ou identificar algo, como a etiqueta de preços no mercado por exemplo. Atributos são informações fornecidas na Tag para que ela se comporte conforme esperado. Existem atributos globais, que podem ser aplicados a todas as Tags, e atributos específicos, que são direcionados a cada Tag, de acordo com a sua especificação. Listas ordenadas e não ordenadas em HTML são estruturas de dados que podem ser usadas para organizar informações de um documento. A diferença entre as duas é a forma como os itens são apresentados.
